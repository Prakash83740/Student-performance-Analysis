📊 Student Performance Analysis (Data Analyst Project)



📌 Project Overview



This project focuses on analyzing student performance data to identify key factors affecting academic scores. The goal is to extract meaningful insights and support data-driven decision-making.



\---



🎯 Business Problem



Educational institutions want to understand:



\- What factors impact student performance?

\- How preparation and background affect scores?

\- How to improve student outcomes?



\---



📊 Dataset Description



The dataset contains the following features:



\- Gender

\- Parental Level of Education

\- Lunch Type

\- Test Preparation Course

\- Math Score

\- Reading Score

\- Writing Score



\---



🔍 Key Analysis Performed



\- Data Cleaning \& Preprocessing

\- Handling Categorical Variables

\- Exploratory Data Analysis (EDA)

\- Correlation Analysis

\- Feature Impact on Scores



\---



📈 Key Insights



\- Students who completed test preparation scored higher 📊

\- Standard lunch students performed better than free/reduced

\- Parental education has a positive impact on scores

\- Female students performed better in reading \& writing

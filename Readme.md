# 📊 Student Performance Analysis Dashboard

## 📌 Project Overview
This project analyzes student performance using **Python and Power BI** to understand how different factors such as gender, parental education, lunch type, and test preparation affect academic scores.

The goal is to identify key insights from data and present them in an interactive dashboard.

---

## 📁 Dataset Information
The dataset includes the following features:

- Gender  
- Parental level of education  
- Lunch type  
- Test preparation course  
- Math score  
- Reading score  
- Writing score

## 🔍 Key Insights
- Students who completed test preparation scored higher on average  
- Standard lunch students performed better than free/reduced lunch group  
- Parental education level has an impact on student performance  
- Strong correlation exists between math, reading, and writing scores
